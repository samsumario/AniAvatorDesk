# AniAvatorDesk
This is a desktop mascot app that loads and displays 3D models in VRM format.

## Installation Guide
- node.js : Download from https://nodejs.org/en/download
- electron.js : Run @ CLI
  ```sh
  npm i electron -g
  ```
- Save VRM models to the model folder

## Libraries
- three.js
- three-vrm

## How to Launch
Run @ CLI
  ```sh
  electron main.js
  ```
